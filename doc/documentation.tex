\begin{abstract}
libgdbc ist eine library welche die Verbindung bzw Kommunikation mit einem gdbserver verwalten soll.
Hauptziel dabei ist es die bestehende Implementierung im radare2 Projekt zu ersetzen.
\end{abstract}
